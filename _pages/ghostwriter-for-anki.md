# Ghostwriter for Anki

**Ghostwriter for Anki** is a browser extension that helps you turn what you’re reading into high-quality Anki cards—fast. It includes an AI “Copilot” for drafting Q/A, a triage flow for reviewing cards before they’re sent, and one-click export to Anki.

## What it does
- **Copilot suggestions while you type** (Tab to accept completions)
- **Smart generation from selected text on any page**
- **Triage queue** to accept/reject quickly before anything is exported
- **Outbox + undo** for safety and review
- **Optional graph dashboard** for exploring your collection

## How it works
1. Select text (or write your own prompt) and generate draft cards.
2. Review in **Triage** (Accept / Reject / Next).
3. Send accepted cards to Anki via **AnkiConnect**.

## Requirements
- Desktop **Anki** installed and running
- **AnkiConnect** add-on installed (ID 2055492159): https://ankiweb.net/shared/info/2055492159

## Setup (2 minutes)
1. Install from the Chrome Web Store.
2. Open the extension **Options** and choose an AI provider (or disable AI features).
3. Paste your API key (stored locally in the extension options).
4. Ensure Anki is open and AnkiConnect is installed, then click “Refresh decks/types” inside the extension.

## Privacy
- You control the AI provider and API key.
- When you use AI features, the text needed to generate suggestions is sent to the selected provider.
- No account required.
